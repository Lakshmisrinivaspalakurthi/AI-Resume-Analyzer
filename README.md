AI Resume Screening System
An AI-powered candidate ranking system that matches resumes with job descriptions using NLP and Sentence Transformers.

Features
- Upload multiple resumes
- Upload/Paste Job Description
- Semantic Similarity Matching
- Candidate Ranking
- ATS-style Scoring
 
Tech Stack
- Python
- Streamlit
- NLP
- Sentence Transformers
- Scikit-Learn

Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
```
